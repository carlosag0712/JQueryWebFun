.click function

In an nutshell, triggers an action once the user clicks on an element, could be text, buttoms, paragraphs, etc.

Example:

Click the following button to change this text to blue, click on this text to set it back to grey

.hide & .show functions

These are a really cool functions to manipulate how things appear and dissapear on the screen. This could be useful to creat cool animations

Example:

Click the "hide" button to hide this text. click on the "show" button to bring it back.

.toggle function

Toggle function allows you to switch the visibility of an element. it is like a combination of .hide and .show when used in its most basic form

Example:

Click the "toggle" button to make this text appear and dissapear

.slideUp and .slideDown functions

Switches the visibility of an item upwards or downwards. Seems useful if you have a condition which an element should elegantly appear or dissapear.

Example:

Click the "Slide Up" and "Slide Down" buttons to make this text move upwards and downwards

.slideToggle function

Switches the visibility of an item upwards or downwards with one single command. Useful over .slideUp and .slideDown separately if you need the element to do both.

Example:

Click the "Slide Up & Down" button to make this text go upwards and downwards

this is very useful when part of you logic entails to change the appearance of an element say, in an ecommerce site, an item is out of stock and you want to highlight it differently. You can add a class that will in turn call specific css styling for this.


Example:

If you click the "Add Class" button, the following actions will be triggered: the "p" tag containing this paragraph will be given the class name "addClass" which will trigger the following css code: " .addClass{ color: red; background: grey; }"
